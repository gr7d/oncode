# OnCode
A very, very lightweight code editor.


## Features
* Highlighting

* Suggestions

* Auto intending


## Supported languages
- Javascript 

- HTML

- CSS

## Usage
```javascript
const editor = new OnCode(
  document.querySelector("#myEditor"), // Targeted element
  "js" // Programming language; Supported values: "js", "html", "css"
);
```
